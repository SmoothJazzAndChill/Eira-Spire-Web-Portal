HTML Formatting Using HTML in Text Box The fields that can be filled in using a multi-line textbox support HTML tags to allow you to add more expression to the content. Below I have outlined a few that might be useful, however it's not limited to these. Click here to see a full list of tags. Unfortunately, you cannot use custom CSS or style attributes
Paragraph <p>Text goes here</p>
Linebreak <br>
Images <img src="IMAGE URL" width="100px" height="100px">
Bold <b>Text in here is bold</b>
Underline <u>Text in here is underlined</u>
Italics <i>Text in here is in italics</i>
Links <a href="URL" target="_new">Link</a>
Lists <ul>
  • <li>List Item</li>
</ul>
<ol>
  1. <li>List Item</li>
</ol>
Div <div>Other HTML goes in here</div>
Tables <table><tr> </tr> <tr> </tr>
<th>Heading 1</th> <th>Heading 2</th>
<td>Data 1</td> <td>Data 2</td>
</table>
